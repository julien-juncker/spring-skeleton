This project allows you to quickly set up a Spring Boot stack with the following technologies:
- Spring Boot 3.4.4
- Flyway
- Java 21
- MapStruct
- Karate Labs
- OpenAPI Generator
- TestContainers

This project follows Domain-Driven Design (DDD) principles.
